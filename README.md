# Quality Enterprises AI Readiness Intake

Static intake form for Quality Enterprises AI planning conversations.

## Hosting

This site is intended for GitHub Pages.

## Form Delivery

Submissions post to FormSubmit and are emailed to `bradd@gamma.tech`.

The first live submission usually triggers a FormSubmit activation email to the recipient. After activation, future submissions should be delivered normally.

## Local Preview

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.
